MATLAB Language 使用2D FFT過濾


例

與1D信號一樣，可以通過應用傅立葉變換，乘以頻域中的濾波器並轉換回空間域來過濾圖像。以下是使用Matlab將高通濾波器或低通濾波器應用於圖像的方法：

讓image成為原始的未過濾圖像，這裡是如何計算其2D FFT： 

ft = fftshift(fft2(image));

現在要排除部分光譜，需要將其像素值設置為0.原始圖像中包含的空間頻率從中心映射到邊緣（使用fftshift ）。要排除低頻，我們將中心圓形區域設置為0。

以下是使用內置函數生成半徑為D的圓盤形二進制掩碼的方法： 

[x y ~] = size(ft);
D = 20;
mask = fspecial('disk', D) == 0;
mask = imresize(padarray(mask, [floor((x/2)-D) floor((y/2)-D)], 1, 'both'), [x y]);

屏蔽頻域圖像可以通過將FFT逐點與上面獲得的二進制掩碼相乘來完成： 

masked_ft = ft .* mask;

現在，讓我們計算逆FFT： 

filtered_image = ifft2(ifftshift(masked_ft), 'symmetric');

圖像中的高頻是銳邊，因此該高通濾波器可用於銳化圖像。
